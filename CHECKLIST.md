# Download Checklist

Track progress of downloading legacy folders from orthodox.net.
Use `archive_folder.php` to download one folder at a time.

## How to download a folder

1. Visit `https://www.orthodox.net/archive_folder.php?folder=FOLDERNAME`
2. Save the zip to your Downloads folder
3. Extract into `archive/public_html/` in this repo
4. Update this checklist
5. Commit and push when ready
